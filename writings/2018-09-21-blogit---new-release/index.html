blogit - new release

blogit.py is the engine that powers this humble blog. It's written in Python, and like all my software development efforts it strives to be minimal with two aspects. The first, include the smallest number of features that make the software useful, but not obsolete. The second aspect of minimalism is as little code as possible. This version has all the features of the previous versions but it's a bit less code:

A hugh thanks goes to Boudhayan Gupta who did the updates to the dependencies and setup.py.

Give blogit a spin, I hope you like it!
